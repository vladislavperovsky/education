## Описание системы ostrovok.ru
### API Gateway

| Поле | Значение |
|--------|--------|
| Название системы | API Gateway |
| Категория | System of Engagement (SoE) |
| Роль в бизнесе | Единая точка входа для клиентских и партнерских приложений. Выполняет маршрутизацию запросов, авторизацию и управление доступом к внутренним сервисам платформы. |
| Входящие потоки данных | REST-запросы от Веб/Мобильного приложения, REST-запросы от B2B API Portal |
| Исходящие потоки данных | REST-запросы в Search Service, Booking Service, Customers & Orders Service, Loyalty Service |
