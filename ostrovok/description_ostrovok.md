## Описание системы ostrovok.ru

### API Gateway

| Поле | Значение |
|--------|--------|
| Название системы | API Gateway |
| Категория | System of Engagement (SoE) |
| Роль в бизнесе | Единая точка входа для клиентских и партнерских приложений. Выполняет маршрутизацию запросов, авторизацию и управление доступом к внутренним сервисам платформы. |
| Входящие потоки данных | REST-запросы от Веб/Мобильного приложения, REST-запросы от B2B API Portal |
| Исходящие потоки данных | REST-запросы в Search Service, Booking Service, Customers & Orders Service, Loyalty Service |

### Search Service

| Поле | Значение |
|--------|--------|
| Название системы | Search Service |
| Категория | System of Record (SoR) |
| Роль в бизнесе | Выполняет поиск доступных вариантов размещения, агрегирует предложения от поставщиков и формирует результаты поиска для клиентов. |
| Входящие потоки данных | REST-запросы поиска от API Gateway |
| Исходящие потоки данных | SOAP/XML запросы в Amadeus, REST/JSON запросы в Expedia, REST/XML запросы в системы прямых отелей, выгрузка данных в DWH |

### Booking Service

| Поле | Значение |
|--------|--------|
| Название системы | Booking Service |
| Категория | System of Record (SoR) |
| Роль в бизнесе | Управляет жизненным циклом бронирования: создание, изменение, отмена и получение статуса бронирования. |
| Входящие потоки данных | REST-запросы от API Gateway |
| Исходящие потоки данных | REST/SOAP запросы поставщикам (Amadeus, Expedia, Отели), выгрузка данных в DWH |

### Customers & Orders Service

| Поле | Значение |
|--------|--------|
| Название системы | Customers & Orders Service |
| Категория | System of Record (SoR) |
| Роль в бизнесе | Централизованное хранение данных о клиентах, заказах и их статусах. Управляет историей заказов и профилями пользователей. |
| Входящие потоки данных | REST-запросы от API Gateway, данные о бонусах от Loyalty Service, подтверждения уведомлений от Notification Service |
| Исходящие потоки данных | REST-запросы в Loyalty Service, REST-запросы в Notification Service |

### Notification Service

| Поле | Значение |
|--------|--------|
| Название системы | Notification Service |
| Категория | Внешний сервис |
| Роль в бизнесе | Отправка клиентам уведомлений о статусах заказов, оплат и бронирований через Email, SMS и Push-каналы. |
| Входящие потоки данных | Запросы на отправку уведомлений от Customers & Orders Service |
| Исходящие потоки данных | Email-сообщения, SMS-сообщения, Push-уведомления, статус доставки в Customers & Orders Service |

### Payment Gateway

| Поле | Значение |
|--------|--------|
| Название системы | Payment Gateway |
| Категория | Внешняя система |
| Роль в бизнесе | Обработка платежей пользователей и взаимодействие с банковскими платежными системами. |
| Входящие потоки данных | Данные из DWH, результаты антифрод-проверок от Fraud Detection, статусы подтверждения отелей |
| Исходящие потоки данных | Финансовые данные в DWH, запросы на антифрод-проверку в Fraud Detection, запросы подтверждения бронирования в Отель |

### Fraud Detection

| Поле | Значение |
|--------|--------|
| Название системы | Fraud Detection |
| Категория | System of Insight (SoI) |
| Роль в бизнесе | Выполняет анализ платежных операций для выявления мошеннических действий и предотвращения финансовых рисков. |
| Входящие потоки данных | Данные о платежных транзакциях от Payment Gateway |
| Исходящие потоки данных | Результаты проверки транзакций в Payment Gateway |

### DWH

| Поле | Значение |
|--------|--------|
| Название системы | DWH (Data Warehouse) |
| Категория | System of Insight (SoI) |
| Роль в бизнесе | Централизованное хранилище исторических данных платформы. Консолидирует данные о поисках, бронированиях и платежах для аналитики и отчетности. |
| Входящие потоки данных | Выгрузки данных из Search Service, Booking Service и Payment Gateway |
| Исходящие потоки данных | Витрины данных и аналитические выборки для BI/Analytics |

### BI / Analytics

| Поле | Значение |
|--------|--------|
| Название системы | BI / Analytics |
| Категория | System of Insight (SoI) |
| Роль в бизнесе | Формирование аналитической отчетности, расчет бизнес-показателей и поддержка принятия управленческих решений. |
| Входящие потоки данных | Аналитические витрины и исторические данные из DWH |
| Исходящие потоки данных | Дашборды, отчеты, аналитические показатели для бизнеса |

### B2B API Portal

| Поле | Значение |
|--------|--------|
| Название системы | B2B API Portal |
| Категория | System of Engagement (SoE) |
| Роль в бизнесе | Предоставляет партнерам доступ к функциональности платформы через программный интерфейс. |
| Входящие потоки данных | REST-запросы от агентств, ответы от API Gateway |
| Исходящие потоки данных | REST-запросы в API Gateway, ответы агентствам |

### Внешний поставщик размещения

| Поле | Значение |
|--------|--------|
| Название системы | Amadeus / Expedia / Прямые отели |
| Категория | Внешняя система |
| Роль в бизнесе | Предоставляет информацию о наличии номеров, тарифах и принимает бронирования. |
| Входящие потоки данных | Запросы поиска и бронирования от Search Service и Booking Service |
| Исходящие потоки данных | Данные о доступности номеров, ценах и статусах бронирования |

### Отель

| Поле | Значение |
|--------|--------|
| Название системы | Отель |
| Категория | Внешняя система |
| Роль в бизнесе | Подтверждает или отклоняет бронирование, обеспечивает фактическое оказание услуги размещения. |
| Входящие потоки данных | Запросы подтверждения бронирования от Payment Gateway |
| Исходящие потоки данных | Статусы подтверждения или отмены бронирования в Payment Gateway |
